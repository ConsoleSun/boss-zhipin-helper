加载扩展

FAQ

常见问题

支持哪些招聘平台？
目前支持 BOSS直聘（boss.cn / zhipin.com）和 智联招聘（zhaopin.com / zhaopin.cn）。选择器系统支持多级回退，网站改版时大部分功能可自愈。
MiMo 和 DeepSeek 如何选择？
DeepSeek：响应快速，15秒内返回，稳定性好，推荐日常使用。
MiMo 2.5：推理模型，回复质量更高但耗时较长（45秒超时），适合追求回复质量的场景。扩展已对 MiMo 的 reasoning_content 字段做兼容处理，直接使用即可。
两个模型都需要自行申请 API Key。
我的数据安全吗？
绝对安全。所有投递记录、聊天消息、岗位信息、AI 对话历史全部存储在浏览器本地（IndexedDB + chrome.storage.local），数据不会上传到任何服务器。AI API 调用仅发送当前对话上下文，不传输历史数据。
如何更新扩展？
从 GitHub 或 Gitee 拉取最新代码，然后在 chrome://extensions 点击扩展的刷新按钮即可。数据存储在浏览器中，更新不会丢失。
为什么点击按钮没有反应？
请检查：1) 是否已在设置页填写 API Key；2) 当前页面是否为职位详情页或聊天页（职位列表页不会显示 AI 助手面板）；3) 点击右下角 🐛 调试按钮查看日志；4) 确认扩展已正确加载（F12 控制台搜索 [BossHelper]）

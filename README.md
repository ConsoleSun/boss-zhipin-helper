# BOSS 直聘助手

一个 Chrome 浏览器扩展，帮助在 BOSS 直聘上整理求职信息。

## 功能

- **AI 招呼语** — 浏览职位时自动抓取 JD，点一下生成个性化打招呼语
- **AI 对话助手** — 聊天页面弹出侧边面板，粘贴 HR 消息自动生成回复
- **岗位管理** — 自动保存浏览过的职位，支持置顶、搜索、删除
- **简历解析** — 上传 PDF 简历，AI 提取信息用于生成更精准的回复
- **数据面板** — 投递记录、沟通统计，支持 JSON 导出
- **右键发送** — 聊天页面右键 HR 消息直接发送到 AI 助手
- **双模式回复** — 专业模式/拟人模式自由切换
- **双模型支持** — DeepSeek / MiMo 2.5

## 安装

1. 下载代码，解压
2. Chrome 地址栏输入 `chrome://extensions`，开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择项目文件夹
4. 打开 BOSS 直聘即可使用

> 注意：`icons/` 目录下需要 icon16.png、icon48.png、icon128.png 三个图标文件

## 配置

加载扩展后，点击右上角扩展图标 → 设置：

- 填写个人信息（技能、经验等）
- 选择 AI 模型并填入 API Key
- 选择回复风格（专业/拟人）
- 上传 PDF 简历（可选）

## 数据存储

所有数据存储在浏览器本地（IndexedDB + chrome.storage.local），不上传任何服务器。
